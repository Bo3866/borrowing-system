# 校園空間與器材租借系統

## 📋 系統概述

這是一個為校園課指組設計的完整租借管理系統，基於第二章軟體需求規格開發。系統支持學生租借空間和器材，課指組人員進行資源管理和審核。

## 🎯 主要功能

### 1. 借用流程（Epic 1）
- **1.1 線上借用申請** - 學生可直接在線填寫申請表
- **1.2 借用狀態即時查詢** - 實時查看申請進度
- **1.3 活動企劃書數位遞交** - 支持文件上傳
- **1.4 申請案儲存功能** - 保存草稿申請
- **1.6 可用狀態顯示** - 清晰展示資源是否可租
- **1.7 分類搜尋功能** - 按類型和關鍵字搜尋
- **1.8 補件/修改機制** - 支持申請修改
- **1.9 申請狀態變更通知** - 審核結果通知
- **1.13 器材報修回報** - 故障器材報修機制

### 2. 空間與器材管理（Epic 2）
- **2.1 資源資訊與可用性管理** - 完整的資源管理
- **2.2 一鍵核准/退回** - 快速審核機制
- **2.3 自動檢查與鄰近時段鎖定** - 時間衝突檢測
- **2.5 逾期未返與逾規自動催撤** - 自動催缴機制
- **2.6 借用紀錄查詢** - 完整查詢歷史
- **2.7 資源統計分析** - 資源使用分析

### 3. 系統維護（Epic 3）
- **3.1 系統異常與錯誤日誌** - 系統健康監控

### 4. 審核流程（Epic 4）
- **4.1 線上審核機制** - 完整的線上審核面板

## 🚀 快速開始

### 1. 打開系統
直接在瀏覽器中打開 `index.html` 文件

### 2. 默認用戶
- **用戶名**：林宇芊
- **角色**：學生
- 系統已預設登入狀態

### 3. 主要頁面

#### 📊 首頁
- 快速導航卡片
- 系統狀態顯示
- 快捷功能入口

#### 🔍 我要租借
- **篩選資源**
  - 按類型篩選（空間/器材）
  - 按日期篩選
  - 關鍵字搜尋

- **可用資源**
  - 空間：教室A101、多功能會議室、戶外活動草坪
  - 器材：投影機、音響系統、舞臺

- **租借流程**
  1. 查看資源詳情
  2. 點擊"租借此資源"
  3. 填寫申請表單
  4. 提交申請
  5. 等待審核

#### 📋 我的申請
- 查看個人所有申請記錄
- 按狀態篩選（審核中/已核准/已拒絕/已完成）
- 待審核申請可取消

#### 📊 資源管理（課指組人員）
- **資源列表**
  - 查看所有資源
  - 編輯資源信息
  - 啟用/禁用資源

- **統計分析**
  - 總租借次數
  - 本月租借數
  - 平均滿意度
  - 設備故障率

#### ✅ 審核面板（課指組人員）
- **待審核**
  - 查看所有待審核申請
  - 核准或拒絕申請
  - 提供拒絕理由

- **審核紀錄**
  - 查看歷史審核記錄
  - 審核狀態追蹤

## 💾 數據結構

### 資源（Resources）
```javascript
{
    id: 'res001',
    name: '資源名稱',
    type: 'space' | 'equipment',
    description: '説明',
    location: '位置',
    available: true | false,
    capacity: 50 // 只適用於空間
}
```

### 申請（Applications）
```javascript
{
    id: 'app001',
    applicantName: '申請人名稱',
    resourceId: 'res001',
    resourceName: '資源名稱',
    startDate: '2024-05-15T09:00',
    endDate: '2024-05-15T12:00',
    purpose: '使用目的',
    status: 'pending' | 'approved' | 'rejected' | 'completed',
    submitDate: '2024-05-10',
    phone: '電話號碼'
}
```

## 🔌 API 集成點

系統預留以下集成點，可連接實際後端API：

1. **資源查詢** - `GET /api/resources`
2. **申請提交** - `POST /api/applications`
3. **申請查詢** - `GET /api/applications`
4. **審核操作** - `PUT /api/applications/:id/review`
5. **狀態通知** - WebSocket 實時通知

## 📱 使用場景示例

### 場景1：學生提交租借申請
1. 點擊"我要租借"
2. 搜尋想要的教室或器材
3. 點擊"租借此資源"
4. 填寫申請表（申請人、日期、用途）
5. 上傳活動企劃書（如需要）
6. 提交申請
7. 在"我的申請"中追蹤狀態

### 場景2：課指組人員審核申請
1. 登入系統（需管理員權限）
2. 進入"審核面板"
3. 查看"待審核"申請列表
4. 檢查申請詳情
5. 點擊"核准"或"拒絕"
6. 填寫審核意見（如拒絕）
7. 申請人將收到通知

### 場景3：資源管理
1. 進入"資源管理"
2. 切換至"資源列表"標籤
3. 查看所有可用資源
4. 編輯或禁用資源
5. 查看"統計分析"了解使用情況

## 🛠️ 開發者指南

### 模擬不同角色

在 `app.js` 中修改 `currentUser` 以測試不同角色：

```javascript
// 學生角色
currentUser.role = 'student';

// 課指組人員
currentUser.role = 'staff';

// 系統管理員
currentUser.role = 'admin';
```

### 添加新資源

在 `app.js` 中的 `resources` 數組中添加新對象：

```javascript
{
    id: 'res007',
    name: '新資源',
    type: 'space',
    description: '新資源描述',
    location: '位置',
    available: true,
    capacity: 100
}
```

### 模擬申請

在 `app.js` 中的 `borrowApplications` 數組中添加新申請對象進行測試。
