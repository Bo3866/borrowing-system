📋 測試場景

場景1：學生租借流程

  1. 左側導航點擊"我要租借"
  2. 查看可用的空間和器材
  3. 點擊"租借此資源"按鈕
  4. 填寫租借申請表單
  5. 提交申請
  6. 在"我的申請"中查看申請狀態

場景2：課指組審核申請

  1. 選擇"課指組人員"角色
  2. 點擊"審核面板"
  3. 查看待審核的申請列表
  4. 點擊"核准"或"拒絕"
  5. 查看"審核紀錄"標籤歷史

場景3：資源管理

  1. 選擇"課指組人員"角色
  2. 點擊"資源管理"
  3. 在"資源列表"中查看所有資源
  4. 點擊"編輯"修改資源信息
  5. 切換至"統計分析"查看使用統計
